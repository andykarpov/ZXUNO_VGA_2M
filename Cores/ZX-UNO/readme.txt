ZX-UNO, версия T24

Корка представляет собой основную корку, ради которой и задумывался проект ZX-UNO, по-сути.
Поддерживает режимы классического ZX Spectrum 48k с режимом "медленной" памяти, режимы классического 128к спектрума, пентагона, и прочих.
Также в данной корке подключен DivMMC для быстрой и удобной загрузки TAP/TRD файлов с SD-карты через NMI-меню.

Для работы DivMMС на SD-карте требуется присутствие в корне карты файлов из архива ESXDOS_ZX1.rar.

Горячие клавиши данной корки:
ScrollLock         - выбор развертки VGA/RGB
Ctrl+Alt+F5        - NMI для входа в меню DivMMC
Ctrl+Alt+Del       - Soft reset
Ctrl+Alt+Backspace - Hard reset
