В этой директории живет образ 16-мегабайтной SPI FLASH.

Для обновления flash:
0) обеспечьте бесперебойное питание для платы, сообщество рекомендует воспользоваться PowerBank'ом.
1) положите образ FLASH.ZX1 в корень SD-карточки
2) зайдите в BIOS и выберите пункт обновления FLASH
3) дождитесь окончания загрузки.

Также в этой директории Вы найдете файл recovery.bit.
Это специальный bitstream, который можно залить в FPGA через JTAG программатор.
Данный bitstream запустит урезанную версию BIOS, с помощью которого Вы сможете вернуть плату к жизни после, например, 
неудачных обновлений путем ее окирпичивания :)

ЗЫ: recovery.bit можно залить даже Altera USB Blaster'ом, как оказалось.
Для этого нужно воспользоваться софтинкой под названием urjtag (есть под windows, linux, raspberry pi, и тп).
